Anleitung:

Zunächst muss der Server gestartet werden.
Navigieren Sie in der Konsole/Terminal zu /Final_Project/Server und führen Sie den befehl "node app" aus.
Der Server startet nun und ist einsatzbereit sobalt die Zeile "server status : online " zu lesen ist.
Nun kann die Datei "index.html" im Verzeichnis "/Final_Project/lsf" mit dem Browser geöffnet werden.
Es sollte nun die Webseite mit den Visualisierungen zu sehen sein.
